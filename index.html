2
STEP 02
Train on Google Teachable Machine (free, browser-based)
Go to teachablemachine.withgoogle.com → New Project → Image Project → Standard model.

Create classes matching your sample type, e.g. for Water: "Algae", "Bacteria", "Chemical", "Safe".

Upload your downloaded images into each class (60–100 per class). Click Train. Takes 5–15 min in your browser — no GPU, no cost.

Export → TensorFlow.js → Upload (shareable link) → copy the URL.
teachablemachine.withgoogle.com →

After exporting from Teachable Machine, paste each model URL here. The app will use your real AI model instead of the simulation.
